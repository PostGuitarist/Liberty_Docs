## Welcome to the O'Malley Trilogy Documentation

This documentation will cover how to get by in most of the O'Malley Trilogy classes using a M1(+) MacOS device.

This documentation is **NOT** guaranteed to work for Dr. O'Malley's classes. He has not tested this documentation and has not approved it. This is a community-driven project and is not officially supported by Dr. O'Malley.
