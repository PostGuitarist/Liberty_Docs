# ERD's for Databases

## Pre-requisites

- [UTM](https://mac.getutm.app/)
    - Windows 11 ARM Installed

## Installation

Inside of your Windows 11 VM, open Edge and go to [ER-Assistant](http://highered.mheducation.com/sites/0072942207/student_view0/e_r_assistant.html)

1. Click on the `ER Assistant` download button
2. Install the application
